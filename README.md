# Monopoly eBanking

This is just a pet project to speed up the game progression.

## MVP

### Before game

- Add player
- Remove player
- Update player
- Start a new game

### During game

- Record cashflow
- Record time
- Display Dashboard
- Easy to input event UI

### After game
- Reset game
- Save data
